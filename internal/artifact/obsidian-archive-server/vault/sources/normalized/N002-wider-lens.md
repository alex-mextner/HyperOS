## Digest

This source broadens the comparison beyond mobile operating systems. It covers Android AppFunctions, Apple App Intents, MCP, local-first databases, CRDTs, capability-oriented collaboration, malleable software, personal knowledge environments, observability tools, and community mobile operating systems.

The current model treats these as interface and data-ownership prior art. AppFunctions, App Intents, and MCP demonstrate structured action publication but do not provide a universal security or transaction model. Automerge, Loro, and related systems inform selected collaborative data types rather than replacing transactional state machines. Malleable-software work informs product composition, while native Agent OS contracts remain versioned and capability-gated.
