# Naming: решение (11.07.2026)

## Принято
**Кодовое имя проекта: Agent OS.** Используется в документации, спеках и репозитории вместо HyperOS (коллизия с Xiaomi). Статус: рабочее кодовое имя на обкатке, не финальный публичный бренд.

## Почему это честное кодовое имя
Agent OS называет корень системы прямо: третья аксиома базиса (всё в графе доступно агентам) — то, что отличает проект от платформенных opt-in решений. Произносится тривиально на обоих языках, объяснений не требует.

## Что держать в голове до публичности
- Как **публичный** бренд «Agent OS» слаб юридически и поисково: словосочетание описательное (генерик), TM-защита почти невозможна, выдача забита (Microsoft «Desktop AgentOS» UFO², академический AIOS, россыпь agentos-проектов). Для кодового имени это не важно; для публичного запуска — важно.
- Основной кандидат публичного имени: **Conatus** (Спиноза; реестры чистые, tech-выдача пустая, фонетических соседей в классе 9 нет). Перед фиксацией — TMview, классы 9/42, EUIPO+WIPO.
- Резерв: Rebro, Morphe, Eidos, Meron, Entos, Insula, Castrum, Cambium.
- **Verta — отклонено** (11.07.2026): конфликт с Vertu (люксовые телефоны, класс 9, живой бренд — AlphaFold с Hermes Agent 05.2026, Agent Q «AI Agent Phone»; одна буква разницы, покупатель решит, что «Verta OS» — ОС от Vertu). Плюс экс-Verta.ai (MLOps, поглощена Cloudera 2024) — возможный живой знак в классе 42.

## История раундов
1. Раунд 1: критерии, процесс, мицелий-семейство → мицелий отклонён.
2. Раунд 2: fluid/hyper/fresh/memex/entity/agentic + автоскрининг 19 имён → FluidSpace/Fluidum → fluid-коннотация отклонена.
3. Раунд 3: квинтэссенция (базис из 3 аксиом), кость/Рим → Trabecula → костная коннотация отклонена.
4. Раунд 4: живые структуры (Cambium/Castrum/Insula) + суть методов обкатки и TMview → сложное произношение.
5. Раунд 5: математика/графы/философия с фильтром произносимости → Verta + Conatus → Verta снята (Vertu).
6. Решение: кодовое имя **Agent OS**, публичное имя — выбрать после обкатки (основной кандидат Conatus).
